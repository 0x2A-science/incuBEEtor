# incuBEEtor bee queen incubator
ESP32 based temperature and humidity control for a bee queen incubator

## PARTS LIST
- ESP32 Dev Board
- 12V power supply
- 4 channel relais module
- 12V silent fan
- 2 heating mats
- rotary encoder
- SSD1306 OLED display
- wires 'n' stuff


## ESP32 PIN REFERENCE
- Heater upper    12
- Heater lower    13
- Fan             14
- Encoder Button  27
- Encoder DT      26
- Encoder CLK     25
- SSD1306 SDA     21
- SSD1306 SCK     22
- HDC1080 SDA     21 
- HDC1080 SCK     22
